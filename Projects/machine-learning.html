Les Misérables Text Prediction

×

Developed a Convolutional Neural Network (CNN) model to predict the next words in text sequences using Victor Hugo's "Les Misérables" as the training dataset. The model learns patterns from the literary text and generates coherent word completions for given phrases.

Project Features

  • CNN Architecture: Implemented Convolutional Neural Network for text sequence prediction
  • Literary Dataset: Trained on Victor Hugo's "Les Misérables" novel
  • Text Generation: Generates coherent word completions for given phrases
  • Performance Metrics: Achieved loss of 2.9492 with 80 training steps
  • Real-time Generation: Produces multiple text samples with varying contexts
  • Google Colab: Complete implementation available in Jupyter notebook

Stroke Prediction Model

×

Developed a comprehensive stroke prediction model using ensemble machine learning techniques including Random Forest, XGBoost, and Decision Tree algorithms. The model analyzes medical data to predict stroke risk with high accuracy, providing valuable insights for healthcare decision-making.

Model Performance

  • Random Forest: 94% accuracy with precision of 0.96 and recall of 0.93
  • XGBoost: 94% accuracy with precision 0.96/0.93, recall 0.93/0.96, F1-score 0.94/0.94
  • Decision Tree: 91% accuracy with precision 0.89/0.93, recall 0.93/0.89, F1-score 0.91/0.91
  • Ensemble Approach: Combined multiple algorithms for robust predictions
  • Medical Features: Analyzed age, gender, hypertension, heart disease, and lifestyle factors
  • Risk Assessment: Binary classification for stroke prediction (0: No stroke, 1: Stroke)

Technical Implementation

  • Data Preprocessing: Handled missing values and categorical variables
  • Feature Engineering: Created meaningful features from medical data
  • Model Training: Used train-test split with 100 estimators for Random Forest
  • Hyperparameter Tuning: Optimized model parameters for best performance
  • Cross-Validation: Ensured model reliability across different data samples
  • Performance Metrics: Comprehensive evaluation using precision, recall, F1-score

Healthcare Recommendations

Based on the feature importance analysis, age and glucose level are high determinants of stroke risk, making them critical factors for healthcare providers to monitor closely. The model's high accuracy (94%) makes it suitable for early screening to identify at-risk patients before symptoms appear, enabling proactive intervention strategies. Healthcare systems should implement targeted preventive care protocols focusing on these key risk factors, particularly for elderly patients and those with elevated glucose levels, while allocating resources efficiently to patients with the highest risk scores. Regular re-evaluation of risk scores should be conducted as patient conditions change, allowing for dynamic adjustment of treatment plans and monitoring protocols based on evolving risk profiles.
